# wowsims-gear-extractor
This is a little Python script that extracts itemids from a json file from WowSims.
## Usage
1. Download `wowsims-gear-extractor.exe` from the newest release.
2. Go to your wowsims sim and enter your gear
3. Hover over Export -> Press Json -> Press Download
4. Make sure both these downloaded files are in the same directory (folder)
5. Open powershell in that directory (right click menu -> open in terminal)
6. Run the program by writing `.\wowsims-gear-extractor.exe .\wowsims.json`
7. Profit
